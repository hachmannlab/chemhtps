chemhtps.misc.mksubdir_struct(dir, max_n_entries=10000, run_always=False)¶

This function takes the content of a dir and makes numbered substructure dirs with each n_entries of the original dir. The motivation was to have a function with limits the number of entries in a directory to a certain threshold (e.g., 10,000 or 30,000) in order to avoid performance issues with the OS/filesystem.

param str dir:Path of the relevant directory
param int max_n_entries:
 Max entries in a subdir
param bool run_always:
 whether to run even if there are less than :param:’max_n_entries’ in :param:’dir’
